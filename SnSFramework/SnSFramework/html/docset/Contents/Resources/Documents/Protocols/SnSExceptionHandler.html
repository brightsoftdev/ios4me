SnSExceptionHandler Protocol Reference

Declared in SnSApplicationController.h

Overview

Receives all the framework exceptions in a single place.

For each method, the return boolean indicates whether the exception has been handled by the protocol, and the last boolean pointer, whether the life cycle of the underlying entity should be resumed or stopped.
