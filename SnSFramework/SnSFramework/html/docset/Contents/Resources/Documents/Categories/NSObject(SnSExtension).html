Declared in NSObject+SnSExtension.h
NSObject+SnSExtension.m

Overview

Defined in order to augment the native NSObject class.

Because the framework extends a class via a category, the project which uses the resulting library must use the -ObjC and -all_load linking option, as explained at [http://developer.apple.com/mac/library/qa/qa2006/qa1490.html](http://developer.apple.com/mac/library/qa/qa2006/qa1490.html) and [http://www.dribin.org/dave/blog/archives/2006/03/13/static](http://www.dribin.org/dave/blog/archives/2006/03/13/static)_objc_lib/ and [http://www.duanefields.com/2008/08/25/objective-c-static-libraries-and-categories/.](http://www.duanefields.com/2008/08/25/objective-c-static-libraries-and-categories/.)
